Detailed Description

Chunk Delivery API - Higher Abstraction.

The Chunk Delivery HA provides the primitives for effective chunks exchange with other peers.
This is a part of the Data Exchange Protocol which provides high level abstraction for sending chunks to a target peers. See chunk_sending_test.c for an usage example
